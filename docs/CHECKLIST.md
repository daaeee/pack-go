# Pack&Go — Чек-лист для разработчиков

## Frontend

### Общие требования
- [ ] Адаптивная вёрстка: десктоп 1440px
- [ ] Все цвета и отступы — из Design Tokens
- [ ] Шрифт: Inter (подключить из Google Fonts)

### Страница «Вход»
- [ ] Валидация email (формат)
- [ ] Валидация пароля (мин. 6 символов)
- [ ] Сообщения об ошибках под полями
- [ ] Ссылка «Нет аккаунта?» → страница регистрации
- [ ] Ссылка «Забыли пароль?» → страница восстановления

### Страница «Регистрация» 
- [ ] Все поля обязательные
- [ ] Проверка совпадения паролей
- [ ] Ссылка «Уже есть аккаунт?» → страница входа

### Страница «Забыли пароль» (3 этапа)
- [ ] Этап 1: валидация email, отправка кода
- [ ] Этап 2: проверка 6-значного кода, таймер повторной отправки
- [ ] Этап 3: новый пароль, подтверждение
- [ ] Кнопка «Вернуться ко входу» на всех этапах

### Дашборд
- [ ] Отображение статистики 
- [ ] Список комнат
- [ ] Плавающая кнопка «+ Новая коробка»
- [ ] Приветствие с именем пользователя
- [ ] Карточка текущего переезда (адреса)

### Мои коробки
- [ ] Поиск по содержимому и номеру коробки
- [ ] Фильтры: по комнате, по статусу, по хрупкости
- [ ] Сетка карточек коробок
- [ ] Кнопка «+ Новая коробка» в шапке
- [ ] Пустое состояние «Нет коробок»

### Создание коробки
- [ ] Выбор комнаты (выпадающий список + кнопка «+» для новой)
- [ ] Поле содержимого 

- [ ] Добавление фото (предпросмотр)
- [ ] Чекбокс «Хрупкое»
- [ ] Кнопка «Сохранить» → валидация (содержимое не пустое)
- [ ] После сохранения → переход на QR-код

### QR-код
- [ ] Отображение QR-кода (сгенерированного на бэкенде)
- [ ] Кнопка «Распечатать» (PDF на A4, 2×4 QR)
- [ ] Кнопка «Добавить ещё коробку»
- [ ] Кнопка «Готово» → на дашборд

### Сканер
- [ ] Запрос доступа к камере
- [ ] Рамка для наведения
- [ ] Автоматическое распознавание QR (jsQR)
- [ ] Кнопка «Ввести номер вручную»
- [ ] Результат сканирования: карточка с содержимым, фото, статусом
- [ ] Кнопка «Отметить распакованной» → обновление статуса

### Логистика
- [ ] Таблица коробок с чекбоксами
- [ ] Кнопка «Выбрать все»
- [ ] Кнопка «Изменить статус» → модальное окно с выбором
- [ ] Проверка допустимости перехода статусов
- [ ] Отображение карточек перевозчиков

### Профиль
- [ ] Отображение информации о пользователе
- [ ] Настройки уведомлений (Email, SMS) — переключатели
- [ ] Кнопка «Подтвердить номер» (отправка SMS-кода)
- [ ] Список переездов (активный + архив)
- [ ] Кнопка «Выйти» → модальное окно подтверждения

### Архив
- [ ] Список завершённых переездов (карточки)
- [ ] Поиск по названию или городу
- [ ] Кнопка «Клонировать» → создание нового переезда
- [ ] Пустое состояние «Нет завершённых переездов»
- [ ] Кнопка «Назад» → в профиль

## Backend


### База данных
- [ ] Таблица `users` (id, email, password_hash, phone, phone_verified, name)
- [ ] Таблица `moves` (id, user_id, name, from_address, to_address, status, completed_at)
- [ ] Таблица `rooms` (id, move_id, name, order_index)
- [ ] Таблица `boxes` (id, move_id, room_id, qr_code, content_text, content_photo_url, is_fragile, status)
- [ ] Таблица `status_history` (id, box_id, old_status, new_status, changed_at)
