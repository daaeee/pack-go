## Design Tokens

Все дизайн-токены экспортированы в JSON для лёгкой интеграции:

| Файл | Описание | Пример использования |
|------|----------|---------------------|
| `colors.json` | Цветовая палитра | `--color-primary: #2A5C8F` |
| `typography.json` | Шрифты и размеры | `--font-size-h1: 28px` |
| `spacing.json` | Отступы и радиусы | `--spacing-md: 16px` |

##  Статусная модель коробок

| Статус | Цвет | Код | Допустимые переходы |
|--------|------|-----|---------------------|
| Упакована | Серый (#6C757D) | `packed` | → `with_carrier` |
| У водителя | Синий (#2A5C8F) | `with_carrier` | → `in_transit` |
| В пути | Оранжевый (#E67E22) | `in_transit` | → `delivered` |
| Доставлена | Зелёный (#28A745) | `delivered` | → `unpacked` |
| Распакована | Фиолетовый (#9B51E0) | `unpacked` | (финальный) |

## Черновые наброски и скриншоты

В репозитории есть две дополнительные папки для наглядности:

###  `/html-prototypes/` — черновые наброски страниц

Здесь лежат HTML/CSS файлы, которые были созданы на этапе прототипирования.  
**Они НЕ являются финальной вёрсткой** и не должны использоваться напрямую в коде.

| Что это | Для чего |
|---------|----------|
| Черновые наброски всех экранов | Понять логику переходов и структуру страниц |
| Примеры интерактивных элементов | Посмотреть, как ведут себя выпадающие списки, модальные окна |
